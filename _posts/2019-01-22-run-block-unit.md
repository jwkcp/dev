파이썬 코드 작성 시 아래와 같이 코드 블럭을 지정해두면 vscode 같은 텍스트 에디터나 pycharm 같은 IDE에서 코드 블럭을 구분할 수 있을 뿐만 아니라, 각 블럭 별 실행도 가능하게 된다. (짱편함)

**사용법**

```
#%% code block 1 here
print("Hi! I'm code line 1)

#%% code block 2 here
print("Hi! I'm code line 2)
```
